Reinicio de estilos


Se debe de utilizar en todas las hojas de estilos para reiniciar los estilos por defecto que tienen todos los navegadores


  • *,
  • *:after,
  • *:before{
  • -webkit-box-sizing: border-box
  • -moz-box-sizing: border-box
  • box-sizing: border-box
  • margin: 0
  • padding: 0
  • list-style: none
  • text-decoration: none
  • border: none
  • outline: none
  • color: font-color}
